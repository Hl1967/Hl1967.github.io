Python易错记录

发表于2022-10-17|更新于2024-01-08
|阅读量:

Python

def函数中定义的参数是在函数体中参与运算的参数，并不能作为运行函数的参数，当函数返回的时候返回的终点并不是所定义的参数，（相当于刚开始定义的参数是自变量）

if data 相当于 if bool（data）进行强制转换
最终意义为 if true or if false

函数的多个返回值多个值在 return 后并列，用英文逗号间隔开

成员运算符（in 也可用 not in意义取反）：用来判断其中一个个体量是否在 大团体中
格式（’个体’ in ‘整体’）

三元表达式
格式： 值1 if 条件 else 值2
条件：布尔返回值的条件（判断真假）
为真执行1 为假执行2
‘应用于求绝对值和最值’

条件分支判断的时候 需要计算判断效率来确定是否执行分支的嵌套 高效计算才是最重要的

bool(‘’) 和 bool(‘ ‘) 第一个为True 第二个为False第一个是空语句 第二个有一个字符的空格bool(‘’) # False
bool(‘ ‘) # True
bool(‘a’) # True
bool(0) # False
bool(1) # True
bool([]) # False
bool([1]) # True
bool({}) # False
bool({‘a’: 1}) # True

python中 列表正向索引从0开始 反向索引从最后一位位次为 -1

index() 方法，使用 列表.index(元素内容) 的形式，我们可以很方便地获得元素在列表中的索引值 （列表索引值）
列表名称.append（）在末尾添加一个元素
列表名称.insert(索引位置, ‘索引内容’) “此时为索引位置 字列表中需要-1”
列表名称 . pop() 的功能是：返回列表指定位置的元素，并删除这个元素。

pop() 可以传入一个参数，表示要删除的元素的索引，如果不传入，就默认为最后一个元素的索引。

del 列表[索引] 意思就是“删除列表中指定索引的元素”。这里正向和反向索引都是支持的。（与pop不同的是del没有返回）
列表.remove(值) 的方式，我们可以在列表中找到 第一个 指定的值，然后将它删除。

len() 函数用于获取序列的元素个数 “即列表”

分片： 列表[起始索引:结束索引]。比如要获取 students 中第三到第五个元素，也就是索引为 2，3，4 的元素，就要写成 students[2:5]。 “注意+1”

分片可以反向索引

列表元组定义 将“【】”改为（）只用一个功能 就是不可以 “增删改”
就是只有一个元素的元组，在格式上与列表是不同的。仅一个元素 x 的列表写成 [x], 但仅一个元素的元组要在括号内多写个逗号：(x,)

用 list() 函数给数据“解锁”，生成一个相同元素的新列表
students_list = list(students)

zip() 函数和 enumerate() 函数。前者用于将两个相同长度的列表中同位置的元素“绑定”到一起，后者用于将列表中的元素用数字的形式数出来。

拓展：在前面方法复习小节图片中列表方法中，除了 copy()、index()、count() 和 pop() 四个方法会返回对应的结果，其它的方法都是遵照“仅处理原列表，不返回新值”的规则。
